# MinCPP

A C++ Lib with no VCRuntime or CRuntime dependency

# Disclaimer
Do not use this in producation. It's a Pet project.

# Objectives
- Minimal size
- Enhanced control
- Core C++ features (classes, templates etc) kept
- Ultimately: see how far a creative approach can achieve in reimplementing STL features
